# Valentine Proposal Website

This is a simple, static Valentine proposal page.

## How to run locally

Open `index.html` in your browser.

## How to publish on GitHub Pages

1. Commit and push this repo to GitHub.
2. Go to your repo on GitHub → **Settings** → **Pages**.
3. Under **Build and deployment**, set:
   - **Source**: Deploy from a branch
   - **Branch**: `main` / `root`
4. Click **Save**. GitHub will provide a URL like:
   `https://YOUR_USERNAME.github.io/Valentine2026/`

## Replace photos (optional)

The page uses the existing WhatsApp image files already in the repo root.
If you want different names or a new folder, update the image `src` paths
in `index.html`.
